# property-photo-categorizer

A web service that automatically categorizes and selects construction photos of residential properties. It should include features to record and link property location information and descriptive text with each photo.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/property-photo-categorizer.git
cd property-photo-categorizer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
